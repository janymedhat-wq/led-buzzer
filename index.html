// LED-Buzzer Project Code (Example: Arduino)

int buzzer = 8;
void setup() { pinMode(buzzer, OUTPUT); }
void loop() {
  tone(buzzer, 2000); // 2000 Hz
  delay(500);
  noTone(buzzer);
  delay(500);
}

CIRCUIT SCHEMATIC

THE SCHEMATIC SHOWS THE BASIC WIRING, HIGHLIGHTING THE NEED FOR A CURRENT-LIMITING RESISTOR FOR THE LED. SCROLL TO VIEW THE ENTIRE SKETCH.

  • LED Connection: Digital Pin (e.g., D10) $\rightarrow$ Current Limiting Resistor ($\approx 220 \Omega$) $\rightarrow$ LED Anode $\rightarrow$ LED Cathode $\rightarrow$ GND.
  • Buzzer Connection: Digital Pin (e.g., D9) $\rightarrow$ Buzzer (+) $\rightarrow$ Buzzer (-) $\rightarrow$ GND.

KEY LEARNING OUTCOMES

THE FOLLOWING CONCEPTS WERE MASTERED DURING THE DEVELOPMENT AND PROTOTYPING OF THIS EMBEDDED SYSTEM.

LEARNING ELECTRONICS BASICS

MASTERING THE PRINCIPLES OF OHM'S LAW ($V = I \times R$), SERIES CIRCUITS, AND COMPONENT POLARITY. UNDERSTANDING THE NECESSITY OF CURRENT-LIMITING RESISTORS TO PROTECT THE LED.

CONTROL THEORY

IMPLEMENTING SIMPLE OPEN-LOOP CONTROL BY DEFINING A CLEAR SEQUENCE OF STATES (ON/OFF). USING PRECISE `delay()` FUNCTIONS TO MANAGE THE TIME INTERVAL OF THE CONTROLLED OUTPUT.

HARDWARE-SOFTWARE INTEGRATION

BRIDGING DIGITAL CODE AND PHYSICAL REALITY USING COMMANDS LIKE `digitalWrite()` TO TRANSLATE LOGICAL STATES (HIGH/LOW) INTO VOLTAGE SIGNALS THAT DRIVE THE HARDWARE.

BASIC WIRING SYSTEM

GAINING PROFICIENCY IN SAFE AND RELIABLE BREADBOARD WIRING. LEARNING TO MANAGE POWER RAILS (VCC AND GND) AND ENSURING STABLE, CLEAN CONNECTIONS FOR PROTOTYPING.
